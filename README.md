## 📓 映画メモ帳

**[→ 開く](https://jose-mvac.github.io/detective-conan/log.html)**

劇場版29作品の感想をまとめて記録できるサイト。

- ★ 総合評価
- 感想メモ
- ミステリー・アクション・ラブコメ・キャラクター・思い出補正の5項目を五角形レーダーチャートで表示
- ユーザーごとにデータを保存（友人間で共有して使えます）
- 視聴済み作品数・平均評価・ランキングを表示

## 🏆 映画ランキングメーカー

**[→ 開く](https://jose-mvac.github.io/detective-conan/sort.html)**

2作品を対決させて、あなただけの最強ランキングを作るサイト。

- 全作品をトーナメント形式で比較
- 「どちらも同じくらい好き」も選べる
- 最後に自分だけのオリジナルランキングが完成
